# Machine Learning Classification Project

## Overview

This project demonstrates the implementation and evaluation of different machine learning classification algorithms using **Scikit-learn** in Python.

The project consists of three parts:

- Building a Decision Tree classifier using the PlayTennis dataset.
- Comparing the Decision Tree with another classification algorithm.
- Applying a classifier to a real-world dataset from the UCI Machine Learning Repository.

---

## Objectives

- Understand how classification algorithms work.
- Implement machine learning models using Scikit-learn.
- Compare different classifiers.
- Evaluate models using common performance metrics.
- Gain experience working with real-world datasets.

### Dataset
PlayTennis Dataset

### Tasks

- Evaluate the model
- Interpret the obtained results

## Part B – Classifier Comparison

A second classification algorithm was implemented and compared with the Decision Tree.

Example classifiers include:

- K-Nearest Neighbors (KNN)
- Naïve Bayes
- Support Vector Machine (SVM)
- Neural Network

The comparison includes:

- Accuracy
- Classification performance
- Advantages and disadvantages
- Model behavior

---

## Part C – UCI Dataset Classification

A dataset from the **UCI Machine Learning Repository** was selected.

The workflow includes:

1. Dataset description
2. Data preprocessing
3. Train/Test split (80% / 20%)
4. Model training
5. Prediction
6. Performance evaluation

Evaluation metrics include:

- Accuracy
- Confusion Matrix
- Precision
- Recall
- F1-score

Finally, the model's strengths, weaknesses, and observations are discussed.
